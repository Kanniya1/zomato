Zomato - Food Delivery Data Insights

This project focuses on analyzing food delivery operations using Python, MySQL, and Streamlit. The application provides an interactive dashboard to manage and analyze customers, restaurants, orders, deliveries, and payments.

Technologies Used
Python
MySQL
Streamlit
Pandas
Plotly Express
SQL
Key Features
Customer Management
Restaurant Management
Order Tracking
Delivery Analytics
Payment Analysis
Interactive Dashboards
Data Visualization
Business Insights Generation
Project Objective

To improve operational efficiency and customer satisfaction by transforming food delivery data into meaningful insights through interactive analytics and visualizations.
